Ghid de instalare a aplicatiei

1) Trebuie descarcat XAMPP, varianta pe care am folosit-o pentru a creea aplicatia este 3.2.2 cu PHP 7.0.4. 
Link-ul pentru a descarca XAMPP este: https://www.apachefriends.org/download.html

2) Se instaleaza XAMPP.

3) Dupa instalare se pornesc serverele de Apache si MySQL.

4) Urmatorul pas este importarea bazei de date. Se deschide un browser si se acceseaza link-ul "http://localhost/phpmyadmin/".

5) Se apasa pe New si la Database name se scrie "licenta" 

6) Se apasa pe Import si apoi se apasa pe Choose File unde se selecteaza fisierul licenta.sql de pe CD si se apasa GO.

7) Se da extract la arhiva "lr.rar" in fisierul "...\xampp\htdocs\"

8) Se deschide un browser si se acceseaza link-ul "http://localhost/lr/index.php"

9) Pentru a beneficia de toate facilitatiile aplicatiei trebuie sa va logati pentru asta va puteti crea un cont nou sau 
puteti folosii un cont deja creat (username: andrei password: password)
